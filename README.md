# Nova AI

Nova AI is a private web application for novel translation and project management. Built with **Next.js** and **Supabase**, the app aims to provide tools for managing content, organizing data, and integrating machine translation services.

## Features

- **Next.js Framework**: Fast and scalable React framework for modern web development.
- **Supabase Integration**: Used for database management, authentication, and storage.
- **Translation Tools**: Designed for integrating APIs like Google Translate and DeepL.
- **Project Management**: Organize translations into projects, glossaries, and chapters.
- **Tailwind CSS**: Sleek and responsive UI components.

## Tech Stack

- **Frontend**: Next.js (React)
- **Backend**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS
- **Authentication**: Supabase Auth
- **State Management**: Context API or Redux (optional)
